Visão Geral do Projeto

O Planejador de Viagens é uma aplicação front-end desenvolvida como parte do evento NLW Journey, organizado pela escola Rocketseat. Esse projeto foi uma oportunidade de aplicar e expandir minhas habilidades em desenvolvimento web, aprendendo novas técnicas e ferramentas de design e programação.

  • Objetivo

    O objetivo do Planejador de Viagens foi desenvolver uma aplicação que não só tivesse uma interface atraente e funcional, mas que também facilitasse o planejamento de viagens dos usuários, desde a seleção de datas até a organização de itinerários. Utilizando uma abordagem centrada no usuário, o projeto buscou integrar funcionalidades que tornassem o planejamento de viagens uma experiência agradável e sem complicações.

  • Desafios

    Aprendizado Acelerado Participar do evento NLW Journey da Rocketseat proporcionou uma imersão rápida em novas tecnologias e práticas de desenvolvimento. O desafio era absorver e aplicar esses conhecimentos de forma eficaz dentro do tempo limitado do evento.

    Manipulação de Datas com Day.js Integrar a biblioteca Day.js para manipulação de datas foi fundamental para oferecer funcionalidades robustas de planejamento de viagens. O desafio foi garantir que a manipulação de datas fosse precisa e intuitiva para o usuário, evitando erros e confusões na seleção e visualização de datas.

    Design Responsivo e Atraente: Criar uma interface que fosse visualmente atraente e responsiva foi essencial para garantir que o Planejador de Viagens oferecesse uma boa experiência em dispositivos de diferentes tamanhos de tela, desde smartphones até desktops.

  • Processo de Desenvolvimento

    Planejamento e Estruturação: O desenvolvimento começou com a criação de wireframes para mapear a estrutura da aplicação. Essa fase envolveu a definição de como os usuários navegariam pela interface, como escolheriam as datas e como visualizariam seus itinerários.

    Desenvolvimento Front-end: Utilizando HTML para a estrutura básica, CSS para a estilização avançada e JavaScript para a lógica de programação, construí a aplicação de forma a garantir que todos os elementos funcionassem de maneira coesa. O uso de CSS avançado permitiu criar uma interface visualmente atraente e interativa.

    Integração do Day.js: A biblioteca Day.js foi integrada para lidar com a manipulação de datas. Isso permitiu criar funcionalidades como seleção de data de viagem, cálculo de duração e exibição de datas importantes. A integração foi feita de maneira a garantir uma experiência de usuário suave e sem interrupções.

Resultado

O Planejador de Viagens - NWL Journey alcançou seu objetivo de ser uma ferramenta eficiente e intuitiva para planejamento de viagens. A integração da biblioteca Day.js permitiu uma manipulação de datas precisa e confiável, enquanto o design responsivo garantiu que a aplicação fosse acessível e utilizável em qualquer dispositivo. O feedback dos usuários foi positivo, destacando a facilidade de uso e a utilidade do planejador.

Conclusão

O projeto Planejador de Viagens - NWL Journey é um exemplo claro de como a combinação de design centrado no usuário, tecnologia avançada e uma abordagem de desenvolvimento iterativa podem resultar em uma experiência de usuário superior. Este projeto não apenas demonstrou minhas habilidades técnicas em front-end, mas também minha capacidade de criar soluções digitais que realmente atendem às necessidades dos usuários, facilitando suas vidas e melhorando suas experiências de viagem.
