Meus projetos

Confira meus projetos disponíveis, cada um refletindo minha dedicação e resultados significativos. Estou aberto a feedbacks, sinta-se à vontade para conhecer meu trabalho.

  • Bikcraft

    Projeto desenvolvido para demonstrar minhas habilidades em front-end. Utilizando HTML, CSS e JavaScript. O Bikcraft é mais do que apenas um site, é uma jornada visualmente deslumbrante através do mundo do design responsivo, mantendo a experiência do usuário em dispositivos móveis.

  • Hogwards Store

    Projeto inspirado no universo de ficção de Harry Potter, criado para demonstrar minhas habilidades em desenvolvimento front-end, especificamente no uso das propriedades do Flexbox para criar um layout responsivo.

  • Planejador de Viagens

    O projeto NWL Journey foi desenvolvido como uma aplicação front-end, com uma estrutura sólida baseada em HTML, estilização avançada utilizando CSS, e lógica de programação aplicada através de JavaScript. O projeto também integra o uso da biblioteca Day.js para manipulação eficiente de datas, garantindo uma experiência de usuário robusta e dinâmica.
